Como Usar o Gerador de QR Code

Siga estas etapas para criar seu QR Code:

  • Digite o conteúdo: No campo acima, insira uma URL (ex.: https://www.exemplo.com), um texto (ex.: "Bem-vindo!"), um número de telefone, e-mail ou qualquer informação.
  • Gere o QR Code: Clique no botão "Gerar QR Code". O código aparecerá abaixo do botão.
  • Escaneie ou salve: Use a câmera do seu celular ou um aplicativo de leitura de QR Code para escanear. Para salvar, clique com o botão direito no QR Code e selecione "Salvar imagem como...".

Dicas: Certifique-se de que URLs começam com "http://" ou "https://". Teste o QR Code com seu celular para verificar se funciona corretamente.
